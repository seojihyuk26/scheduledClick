Add a timer: right click on top of your link/button, and select FoxyClick.

Remove a timer: refresh the page you set the timer.

Important! Refreshing the page you set the timer will remove the timer! You can have only one timer per page.

Liability: we are not liable, if for some reason the automatic click does not work. Test the clicker before using it to anything important.
